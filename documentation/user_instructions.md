### Asennusohjeet:

Kloonaa lukuvinkkikirjasto paikallisesti haluamaasi kansioon komennolla:

```
git clone git@github.com:tommise/lukuvinkkikirjasto.git
```

Suorita gradlen komennot clean ja build:
```
gradle clean
```
```
gradle build
```

Käynnistä sovellus:
```
gradle run
```
